Telephony Service 

TELEPHONY
```
telephony-service
max-ephones 10
max-dn 3
ip source-address 10.1.1.1 port 2000
auto assign 1 to 3
create cnf-files 
```

The numbers for each one of the PC/IP Phone

```
ephone-dn 1
number 101

ephone-dn 2
number 100

ephone-dn 3
number 102

```

In the End do `create cnf-files ` again to update the number list
